# Snake Game in C++

Classic console-based Snake Game built with C++ using Visual Studio.

## Controls
- **w**: move up  
- **s**: move down  
- **a**: move left  
- **d**: move right  
- **p**: pause/resume the game  
- **x**: exit the game  

## How to Run
- Open the `.vcxproj` file in Visual Studio  
- Build and run the project (`Ctrl + F5`)
